# Student Management System (FastAPI)

A simple **Student Management System** built using **FastAPI** and
**HTML**.\
This project allows users to **add, view, and delete student records**
through a simple web interface.

------------------------------------------------------------------------

## 🚀 Features

-   Add new students using an HTML form
-   View the list of students
-   Delete students from the list
-   FastAPI backend with HTML responses
-   Beginner‑friendly FastAPI project

------------------------------------------------------------------------

## 🛠 Technologies Used

-   Python
-   FastAPI
-   HTML
-   Uvicorn

------------------------------------------------------------------------

## 📂 Project Structure

    Student-Management-System

    ├── Management.py

------------------------------------------------------------------------

## ▶️ How to Run the Project

### 1. Install Dependencies

    pip install fastapi uvicorn python-multipart

### 2. Run the Server

    uvicorn Management:app --reload

### 3. Open in Browser

    http://127.0.0.1:8000
